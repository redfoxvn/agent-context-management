# Skill: Session Handoff

## Use For

Preserve enough context for another session or agent to continue safely.

## Key Moves

- Record current status, scope, and verification state.
- Summarize what changed and why.
- Capture assumptions, decisions, blockers, and risks.
- Record remaining work and concrete next action.
- Keep handoff aligned with actual task state.

## Output

- handoff summary
- changed files
- verification state
- remaining work and next action
- known risks or blockers

## Escalate When

- verification gaps are unclear
- status no longer matches actual state
- another session cannot continue safely from the handoff
