# ACM Config

## Skill Pack

Workflow is provided by the installed ACM skills.

## Memory Paths

| Memory Type | Path |
|---|---|
| Project context | `.acm/project.md` |
| Task history | `.acm/tasks/` |
| Feature truth | `.acm/features/` |
| Architecture truth | `.acm/architecture/` |
| Decisions | `.acm/decisions/` |

## Defaults

- Create optional memory folders only when they contain useful durable content.
- Prefer concise, verified, path-oriented project memory.
- Keep workflow procedures in skills, not `.acm/`.
