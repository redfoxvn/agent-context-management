# Tasks Index

## Purpose

Task folders record task-scoped working state, investigations, verification, and handoffs.

Durable feature behavior belongs in `.acm/features/` when present.

## Task Folder Rule

Use:

```txt
.acm/tasks/[YYYY-MM-DD-task-slug]/
```

Default to the smallest artifact set that preserves task state safely.
